
## Real-Time Collaborative Whiteboard

A lightweight, real-time drawing application that allows multiple users to draw on a shared canvas simultaneously. Built using **Node.js**, **Express**, and **Socket.IO**.

## Features
* **Real-Time Collaboration:** Drawing events are broadcast instantly to all connected users.
* **Session History:** New users immediately see the existing drawing history upon joining.
* **Customizable Tools:**
    * Pick any color using the color picker.
    * Adjust brush size using the slider.
* **Responsive Canvas:** Automatically adjusts to the window size.

## DHrw Tech Stack
* **Backend:** Node.js, Express
* **Real-Time Communication:** Socket.IO
* **Frontend:** HTML5 Canvas API, Vanilla JavaScript, CSS
